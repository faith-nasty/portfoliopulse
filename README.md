# Personal Portfolio Website

A modern, responsive personal portfolio website showcasing projects, skills, work experience, and contact information.

## Framework

Vanilla JavaScript with HTML5 and Tailwind CSS (CDN)

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on mobile, tablet, and desktop
- **Smooth Animations**: Fade-in animations, parallax effects, and smooth scrolling
- **Clean Light Theme**: Professional white/gray color scheme with blue accents
- **Interactive Elements**: Hover effects, animated skill badges, and project cards
- **Sections**:
  - Hero section with animated introduction
  - About Me with profile image and social links
  - Projects showcase with 6 sample projects
  - Skills organized by category (Frontend, Backend, Tools)
  - Work Experience timeline with 4 positions
  - Contact form with validation
- **Mobile Navigation**: Collapsible menu for mobile devices
- **Active Navigation**: Highlights current section while scrolling

## Preview

This is a static HTML website that can be opened directly in any browser.

Simply open `index.html` in your browser to view the portfolio.

## Customization

To personalize this portfolio:

1. **Personal Information**: Update name, title, and bio in `index.html`
2. **Projects**: Replace project details, images, and links
3. **Skills**: Modify skill badges to match your expertise
4. **Experience**: Update work history with your own positions
5. **Images**: Replace placeholder images from Picsum with your own
6. **Social Links**: Update social media links in the About section
7. **Colors**: Modify Tailwind classes to change the color scheme

## Theme

**Clean Light Theme**
- Background: White (#FFFFFF) and Light Gray (#F9FAFB)
- Text: Dark Gray (#111827) for headings, Medium Gray (#4B5563) for body
- Accent: Blue (#3B82F6) for interactive elements
- Cards: White with subtle shadows
- Hover Effects: Smooth transitions with scale and shadow changes

## Technologies

- HTML5 (semantic structure)
- Vanilla JavaScript (animations and interactions)
- Tailwind CSS CDN (styling)
- CSS3 (custom animations and effects)
